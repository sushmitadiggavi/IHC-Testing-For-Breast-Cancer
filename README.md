# Virtual IHC Test for Breast Cancer

## Overview
Virtual IHC Test for Breast Cancer is an AI-powered web application designed to assist in breast cancer diagnosis by analyzing medical images and generating virtual immunohistochemistry (IHC) results. The system helps pathologists and researchers by providing a faster and more accessible preliminary analysis.

## Problem Statement
Traditional IHC testing is time-consuming, expensive, and requires specialized laboratory infrastructure. This project aims to provide a virtual solution that supports early diagnosis and research.

## Features
- User Registration and Login
- Secure Authentication System
- Medical Image Upload
- Virtual IHC Analysis
- Automated Result Generation
- Patient Data Management
- MySQL Database Integration
- Report Storage and Retrieval

## Technologies Used
- Python
- Flask
- SQLAlchemy
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## System Workflow
1. User logs into the system.
2. Uploads breast cancer histopathology image.
3. AI model processes the image.
4. Virtual IHC markers are generated.
5. Results are displayed and stored in the database.

## Project Structure
- Authentication Module
- Image Upload Module
- Analysis Engine
- Result Generation Module
- Database Management

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Configure MySQL database
4. Run the application:
   python app.py

5. Open browser:
   http://localhost:5000
